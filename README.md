## 项目简介

博爱恐同症矫治医院是一个讽刺性艺术网站项目，以反讽现实中存在的"戒同所"为初衷。通过虚构一个"恐同症矫治医院"，模仿正规医疗机构的网站风格，将曾经真实用于性少数群体的"矫正"暴行反转应用于恐同者，揭示恐同的荒谬与危害。

项目旨在支持性少数群体的平等权利，讽刺恐同者。

## 技术栈

- **前端框架**：Next.js
- **样式**：Tailwind CSS
- **图标**：Lucide Icons

## 本地运行

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建
pnpm build

# 生产模式
pnpm start
```
